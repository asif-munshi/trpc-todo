# TRPC - Todo

A Todo App Created with NextJS, TRPC.

## Getting Started

Install dependencies:

```bash
  npm install
  # or
  yarn
```

Run the development server:

```bash
  npm run dev
  # or
  yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
